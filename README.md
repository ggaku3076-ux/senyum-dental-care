# Senyum Dental Care

Website landing page multipage untuk demo klinik gigi lokal kecil-menengah.

## Isi Halaman

- `index.html` — beranda ringkas
- `layanan.html` — layanan klinik
- `dokter.html` — profil dokter
- `harga.html` — harga mulai
- `testimoni.html` — testimoni pasien
- `lokasi.html` — lokasi dan kontak
- `faq.html` — pertanyaan umum
- `style.css` — styling utama
- `script.js` — interaksi navbar, FAQ, dan WhatsApp
- `prd-website-reservasi-klinik-gigi.md` — PRD website

## Cara Buka Lokal

```bash
python3 -m http.server 8000
```

Lalu buka:

```text
http://localhost:8000/
```
